# ⚖️ Estudio Jurídico - Plantilla Profesional con Next.js & Tailwind

Una plantilla moderna, elegante y completamente responsive diseñada especialmente para abogados, estudios jurídicos, consultorios legales o profesionales del derecho. Construida con tecnologías actuales para ofrecer una experiencia premium, rápida y accesible.

## ✨ Características principales

- Diseño profesional, limpio y 100% responsive (mobile-first)
- Animaciones sutiles y elegantes con Framer Motion y Lenis (scroll suave)
- Iconos minimalistas de Lucide React
- Optimización automática de fuentes con `next/font/google`
- Estructura modular con componentes reutilizables
- Listo para deploy en Vercel (o cualquier hosting)
- Alto rendimiento y accesibilidad (Lighthouse 90+)

## 🛠️ Tecnologías utilizadas

- **Framework**: Next.js 16.1.6 (App Router)
- **Estilos**: Tailwind CSS 4
- **Animaciones**: Framer Motion 12 + Lenis 1.3
- **Íconos**: Lucide React 0.563
- **Utilidades**: clsx, tailwind-merge, react-use
- **Fuentes**: Roboto, Lora, Montserrat (optimizadas con next/font)
